# 🎵 RankMyMP3 - Guia de Instalação Rápida

## ⚡ Instalação Super Simples

### 🪟 Windows
1. **Baixe** este projeto (botão verde "Code" → "Download ZIP")
2. **Extraia** para uma pasta (ex: `C:\RankMyMP3`)
3. **Clique duplo** em `install.bat` e siga as instruções
4. **Execute** via atalho na área de trabalho ou `run.bat`

### 🐧 Linux / 🍎 Mac
1. **Baixe** este projeto ou clone: `git clone [URL]`
2. **Abra terminal** na pasta do projeto
3. **Execute**: `chmod +x install.sh && ./install.sh`
4. **Execute**: 
   - **Linux**: Procure "RankMyMP3" no menu de aplicações
   - **Mac/Terminal**: `./run.sh`

## 🎯 Após a Instalação

### Windows
- ✅ Atalho na área de trabalho
- ✅ Arquivo `RankMyMP3.bat` para execução direta

### Linux
- ✅ Entrada no menu de aplicações (Áudio/Vídeo)
- ✅ Ícone personalizado
- ✅ Integração nativa com o desktop

## 📋 Pré-requisitos

### Se você não tem Python instalado:

#### Windows
1. Vá para [python.org/downloads](https://www.python.org/downloads/)
2. Baixe a versão mais recente
3. **IMPORTANTE**: Durante instalação, marque ✅ "Add Python to PATH"

#### Mac
```bash
# Com Homebrew (recomendado)
brew install python

# Ou baixe de python.org
```

#### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install python3 python3-pip
```

## 🆘 Problemas?

### "Python não encontrado"
- Windows: Reinstale Python marcando "Add to PATH"
- Mac/Linux: Instale via gerenciador de pacotes

### "Erro ao instalar dependências"
- Verifique conexão com internet
- Windows: Execute como administrador
- Linux/Mac: Use `sudo` se necessário

### wxPython não instala
```bash
# Ubuntu/Debian
sudo apt install python3-wxgtk4.0

# macOS
brew install python-tk
```

## 🚀 Primeira Execução

1. **Adicione suas pastas** de música: `Arquivo > Configurar Pastas`
2. **Comece a classificar** - o programa irá mostrar pares de músicas
3. **Escolha a melhor** entre cada par
4. **Continue** até todas estarem classificadas!

## 💡 Dicas

- **Feche e reabra** quando quiser - o progresso é salvo automaticamente
- **Use filtros** para encontrar músicas específicas
- **Clique direito** nas músicas para mais opções
- **Tags** ajudam a organizar sua coleção
